# INS
Проект для студентов ТПУ в рамках ИШИТР-плюс. Бот производящий анализ изменений(commit, pull request и т.д.) в системе контроля версий(ориентировочно gitlab), и проводящий их первичное ревью (стиль, очевидные ошибки, антипаттерны) и составляющий отчет. Работа бота будет встроена в CI/CD пайплайн
